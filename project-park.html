Project Overview

The Park Theatre Adaptive Reuse project engaged the challenge of transforming a historic theatre building into a new programmatic use while preserving its cultural and architectural character.

The project demonstrates the intersection of technical documentation skills and performance-driven design thinking — using daylight analysis to inform passive strategies appropriate to the building's existing envelope.

Adaptive Reuse Strategy

The existing theatre structure was analyzed for its adaptive potential — identifying which spatial elements to preserve, which to reconfigure, and where new program could be introduced without compromising historic character.

Daylight & Passive Design

Daylight analysis was conducted to evaluate existing aperture performance and to locate new openings or light wells. Passive design strategies including cross-ventilation, thermal mass, and shading were developed in response to the building's orientation and climate context.

Revit Documentation

The project produced 5+ coordinated Revit documentation sheets including floor plans, sections, elevations, and detail drawings suitable for a schematic-level submission.
