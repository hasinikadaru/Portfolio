Project Overview

CBNA 3300 Monroe was unique in the internship as the only project that required on-site field verification. Two site visits were conducted to measure and document existing conditions at the community bank branch before any Revit modeling began.

This project provided direct experience with the gap between as-built conditions and available documentation — a core competency in renovation and adaptive reuse practice.

Field Verification

Dimensions, door and window locations, structural elements, and spatial relationships were recorded during site visits and translated into a verified existing conditions Revit model.

Ground Floor Redesign

The new ground floor layout was developed in Revit from the verified base, proposing renovations to the branch interior while coordinating with existing structural constraints.
