Project Overview

The Vester Pool House project demonstrates a complete residential drawing set workflow — from initial redline markups through final CAD production and PDF deliverables for permit and contractor use.

Despite being a smaller residential accessory structure, the project required full documentation across all drawing types, providing hands-on experience with the breadth of a standard residential construction document set.

Drawing Set Scope

Documentation included foundation plan with footing details, floor plan with dimensions and notes, roof plan with drainage and framing direction, wall sections showing typical construction assembly, and exterior elevations on all four sides.

Workflow

Work proceeded from project lead redlines through CAD updates, coordination of sheet references, and final PDF preparation for submission.
